## russian_roulette.py

一个简陋的俄罗斯轮盘游戏：
- 6个弹仓，随机放一颗子弹
- 每次开枪（yes）枪膛顺移一格
- 活下来奖金指数增长（10 → 100 → 1000...）
- 死了奖金归零
- 输入 no 随时退出

代码写得很原始，就是图个乐子，变量名拼音、没异常处理、while True 裸奔都正常（狗头）

### 运行方式
```bash
python russian_roulette.py
